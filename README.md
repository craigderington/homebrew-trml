# homebrew-trml

Homebrew tap for [trml](https://trml.app) — terminal access to your Mac/Linux machine from your iPhone.

## Install

```bash
brew install craigderington/trml/trml-daemon
```

Homebrew taps automatically. To tap explicitly:

```bash
brew tap craigderington/trml
brew install trml-daemon
```
